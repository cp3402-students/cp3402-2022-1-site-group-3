    Local Environments

    All team members used Docker for local development. Besides, we used Github to track the version control. We created Wordpress folder using "docker compose up" so that we could work on the theme development and keep updating the Github repository. When changes were made locally to the theme, they could be easily committed to the Github repository, and when changes were made to the site content, an export could be uploaded to GoogleDrive to be shared with other members or upload to our live sites.
